# Movie Library - Личная кинотека

**Автор:** Костя Удальцов

## Описание

Приложение для ведения личной коллекции фильмов с графическим интерфейсом на Python (tkinter). Позволяет добавлять фильмы, фильтровать по жанру и году, а также сохраняет данные в JSON-файл.

## Функциональность

- Добавление фильма с указанием названия, жанра, года выпуска и рейтинга
- Валидация ввода: год — число, рейтинг — от 0 до 10
- Отображение всех фильмов в таблице
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году (точное совпадение)
- Автоматическое сохранение в `movies.json`
- Загрузка данных при запуске

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Сохраните файл `movie_library.py`
3. Запустите приложение:
```bash
python movie_library.py
